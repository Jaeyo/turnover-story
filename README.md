# Turnover Story

> 이직은 항상 축하받을 일이다. 

이직을 하기 위해선 큰 용기와 많은 노력이 필요하기에 과정만으로 충분히 축하받을 일이다.

이직을 성공한 이들에게 박수를 👏

이직에 실패한 이들에게 응원을 🤞

이직을 고민하는 이들에게 용기를 💪
